# 📊 CSV to Excel Converter

A simple Python CLI tool to convert CSV files into Excel (.xlsx) format with basic data cleaning.

---

## 🚀 Features

- Convert CSV → Excel (.xlsx)
- Handle missing values (fill with "N/A")
- Simple CLI interface
- Logging for process tracking
- Clean and modular code structure
